# AI for 30 Days Bootcamp

A practical 30-day roadmap to master AI model building for full-stack developers.

## Overview

This bootcamp transforms you from a developer to a full-stack AI-ready developer. You'll learn to wield AI as a powerful tool in your stack, focusing on practical applications over deep theory.

## Prerequisites

- Solid Python skills (functions, classes, virtual environments)
- Git & GitHub basics
- API knowledge (REST, JSON)
- Command line navigation

## Structure

- **docs/**: Documentation and daily notes
- **src/**: Source code organized by days
- **tests/**: Test files
- **notebooks/**: Jupyter notebooks (if used)

## Getting Started

1. Set up your environment (see docs/day01/setup.md)
2. Follow the roadmap in docs/roadmap.md
3. Complete daily activities and projects

## Requirements

Install dependencies:
```
pip install -r requirements.txt
```
